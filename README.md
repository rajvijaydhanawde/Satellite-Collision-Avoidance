# Satellite-Collision-Avoidance
AI-powered satellite collision avoidance system using Reinforcement Learning and orbital propagation to predict conjunctions and optimize avoidance maneuvers.

It is an intelligent satellite safety platform designed to predict potential orbital collisions and recommend optimal avoidance maneuvers using Artificial Intelligence and Reinforcement Learning techniques.

With the increasing number of satellites and space debris in Earth's orbit, manual collision assessment is becoming difficult and time-consuming.Project automates conjunction analysis, risk assessment, and maneuver planning to enhance space mission safety and operational efficiency.
Features
Collision Risk Detection
Orbital Trajectory Analysis
Reinforcement Learning-Based Maneuver Optimization
Real-Time Satellite Monitoring
Space Debris Impact Assessment
Automated Conjunction Analysis
Visualization of Orbital Paths
Performance Evaluation Metrics
